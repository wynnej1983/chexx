# chexx

Chess mobile app built with ReactNative/ReduxToolkit and dumb AI

<img src="/chexx.png" alt="Chexx"/>

## Installation

    $ yarn

## Usage

    $ yarn start
    Runs packager

    $ yarn ios
    Runs app on ios simulator

    $ yarn android
    Runs app on android simulator

## Tests

    $ yarn test
